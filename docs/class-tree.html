Class Hierarchy

  • pvrtask.Color: The color of the Square stimulus, either red or yellow
  • pvrtask.Experiment: The Experiment starts the experimental or practice Session, contains the Graphical User Interface, can save the data, can show calibration screens, etc.
  • pvrtask.IDGenerator: The ID Generator creates a random string to identify the subjects.
  • pvrtask.MotorTrials: Motor Trials to test Subject's motoric skills
  • pvrtask.Orientation: The orientation of the Landolt stimulus.
  • pvrtask.Position: Spatial information about position and size of the stimuli.
  • pvrtask.Session: The Session shows the instructions and presents the blocks and trials.
  • pvrtask.SessionSword: The Session shows the instructions and presents the blocks and trials.
  • pvrtask.Stimuli: Creates a container for the fixation cross, and the central an peripheral stimuli.
  • pvrtask.StimuliSword: Creates a container for the fixation cross, and the central an peripheral stimuli.
  • pvrtask.Sword: A sWord is not a sword but a scrambled word, or actually a random character string that may contain selected characters or not (logical parameter match).
  • pvrtask.Trial: A Trial shows a central and peripheral stimulus.
  • psychopy.visual._BaseVisualStim: A template for a stimulus class, on which PatchStim, TextStim etc...
